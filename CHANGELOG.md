- Complete rewrite with enhanced features

- ASCII and Unicode character conversion
- Dark/light color scheme support
- Comprehensive step-by-step conversion explanations
- Improved error handling and input validation
- Author attribution: Frank486
- GitHub-ready documentation
- Setup.py for easy installation
- MIT License
